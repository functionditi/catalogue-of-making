EXP 1

BASIC PLANT INTERACTION

Since my project is based on the premise of Living Media Interfaces, it also references previous case studies like Botanicus Interacticus and Interactive Plant Growing. These both factor in human-plant touch, and I wanted to explore the value of these interactions myself and set it up using Arduino physical computing.




→ Trying to create a touch sensor using two post-its lined with wire.

→ Also setting up a circuit that would close once a finger touched it, but would only faintly work when finger was wet.




→ Finally using capacativeSense to create a sensor that would detect touch. Sticking some copper tape in the plant's soil to turn it into a sensor too.

→ Attempting to establish a serial connection with p5.js (here, using a potentiometer).




→ Set up a p5.js connection! Represented each instance of plant-touch with a flurry of digital monstera leaves.




→ Attempting to create a more systematic form of visualisation: longer+harder touch results in a larger circle, lighter+softer touch results in a smaller circle.

EXP 2

TOUCH ME NOTS

Thinking about moving away from digital screens as a vehicle to display the information we get from the plants. Why not see if plants themselves can show us this information? Furthermore, what's a plant that, on touch, responds to you the fastest? Mimosa Pudica, of course.




→ Basic sketches for creating a prototype. Sensors underneath could detect when leaves were open or closed.




→ Digging up a Mimosa pudica plant from the field in front of LASALLE, was very thorny!

→ Almost killed the plant in the uprooting process.




→ Cut off the dead branches, resulting in new growth!
